参数扩展

基本使用

name="John"
echo ${name}
echo ${name/J/j}    #=> "john" (替换)
echo ${name:0:2}    #=> "Jo" (切片)
echo ${name::2}     #=> "Jo" (切片)
echo ${name::-1}    #=> "Joh" (切片)
echo ${name:(-1)}   #=> "n" (右侧开始切片)
echo ${name:(-2):1} #=> "h" (右侧开始切片)
echo ${food:-Cake}  #=> $food or "Cake"

length=2
echo ${name:0:length}  #=> "Jo"

更多: Parameter expansion

STR="/path/to/foo.cpp"
echo ${STR%.cpp}    # /path/to/foo
echo ${STR%.cpp}.o  # /path/to/foo.o
echo ${STR%/*}      # /path/to
echo ${STR%%/*}     # (空)

echo ${STR##*.}     # cpp (extension)
echo ${STR##*/}     # foo.cpp (basepath)

echo ${STR#*/}      # path/to/foo.cpp
echo ${STR##*/}     # foo.cpp

echo ${STR/foo/bar} # /path/to/bar.cpp

STR="Hello world"
echo ${STR:6:5}    # "world"
echo ${STR: -5:5}  # "world"

SRC="/path/to/foo.cpp"
BASE=${SRC##*/}   #=> "foo.cpp" (basepath)
DIR=${SRC%$BASE}  #=> "/path/to/" (dirpath)

替换

%rule删除rule到结束匹配的内容，#rule删除开始到rule匹配的内容，即%删除后缀、#删除前缀。另外，默认为最小匹配，叠字%%、##为贪婪匹配。

CodeDescription
${FOO%suffix}删除后缀
${FOO#prefix}删除前缀
${FOO%%suffix}删除最大匹配后缀
${FOO##prefix}删除最大匹配前缀
${FOO/from/to}替换第一各匹配项
${FOO//from/to}替换所有匹配项
${FOO/%from/to}替换前缀
${FOO/#from/to}替换后缀

操作

STR="HELLO WORLD!"
echo ${STR,}   #=> "hELLO WORLD!" (首字母小写)
echo ${STR,,}  #=> "hello world!" (全部小写)

STR="hello world!"
echo ${STR^}   #=> "Hello world!" (首字母大写)
echo ${STR^^}  #=> "HELLO WORLD!" (全部大写)

默认值

ExpressionDescription
${FOO:-val}$FOO, or val if unset (or null)
${FOO:=val}Set $FOO to val if unset (or null)
${FOO:+val}val if $FOO is set (and not null)
${FOO:?message}Show error message and exit if $FOO is unset (or null)

Omitting the : removes the (non)nullity checks, e.g. ${FOO-val} expands to val if unset otherwise $FOO.

条件语句

条件

[[会返回0(假)或1(真)，其他任意遵守相同逻辑的程序均可以作为条件，详见示例。

ConditionDescription
[[ -z STRING ]]字符串为空
[[ -n STRING ]]字符串非空
[[ STRING == STRING ]]字符串相等
[[ STRING != STRING ]]字符串不等
[[ NUM -eq NUM ]]相等
[[ NUM -ne NUM ]]不等
[[ NUM -lt NUM ]]小于
[[ NUM -le NUM ]]小于等于
[[ NUM -gt NUM ]]大于
[[ NUM -ge NUM ]]大于等于
[[ STRING =~ REGEXP ]]正则匹配
(( NUM < NUM ))数值条件

更多条件

ConditionDescription
[[ -o noclobber ]]如果xx设置开启
[[ ! EXPR ]]非
[[ X && Y ]]与
[[ X || Y ]]或

文件条件

ConditionDescription
[[ -e FILE ]]存在
[[ -r FILE ]]可读
[[ -h FILE ]]符号链接
[[ -d FILE ]]目录
[[ -w FILE ]]可写
[[ -s FILE ]]文件大小大于0
[[ -f FILE ]]普通文件
[[ -x FILE ]]可执行
[[ FILE1 -nt FILE2 ]]1比2更新，或1存在2不存在
[[ FILE1 -ot FILE2 ]]1比2更旧，或2存在1不存在
[[ FILE1 -ef FILE2 ]]相同文件

数组

定义数组

Fruits=('Apple' 'Banana' 'Orange')

Fruits[0]="Apple"
Fruits[1]="Banana"
Fruits[2]="Orange"

数组使用

echo ${Fruits[0]}           # 第一个元素 #0
echo ${Fruits[-1]}          # 最后一个元素
echo ${Fruits[@]}           # 所有元素，空格隔开
echo ${#Fruits[@]}          # 元素个数
echo ${#Fruits}             # 第一个元素长度
echo ${#Fruits[3]}          # 第n个元素长度  
echo ${Fruits[@]:3:2}       # 范围 (3开始，长度2)
echo ${!Fruits[@]}          # 所有元素的键，空格隔开

操作

Fruits=("${Fruits[@]}" "Watermelon")    # Push
Fruits+=('Watermelon')                  # Also Push
Fruits=( ${Fruits[@]/Ap*/} )            # 按正则匹配移除
unset Fruits[2]                         # 删除指定元素
Fruits=("${Fruits[@]}" "${Veggies[@]}") # 合并
lines=(`cat "logfile"`)                 # 从文件中读取

选项

选项

set -o noclobber  # 避免覆盖文件 (echo "hi" > foo)
set -o errexit    # 用于在出错时退出，避免级联错误
set -o pipefail   # 显示隐藏的错误
set -o nounset    #

扩展选项

shopt -s nullglob    # 移除不匹配扩展项 ('*.foo' => '')
shopt -s failglob    # 无匹配扩展时直接报错
shopt -s nocaseglob  # 扩展大小写不敏感
shopt -s dotglob     # 通配符匹配点文件 ("*.sh" => ".foo.sh")
shopt -s globstar    # **匹配零个或多个子目录 ('lib/**/*.rb' => 'lib/a/b/c.rb')

命令

CommandDescription
history查看历史

表达式

ExpressionDescription
!$展开最新命令的最后一个参数
!*展开最新命令的所有参数
!-n展开第 n个最近参数
!n展开第 n个参数
!<command>展开命令<command>的最新调用

操作

CodeDescription
!!再次执行上一条命令
!!:s/<FROM>/<TO>/在最新命令中将第一次出现的<FROM>替换为<TO>
!!:gs/<FROM>/<TO>/在最新命令中将所有的<FROM>替换为<TO>
!$:t仅从最近命令的最后一个参数扩展基本称
!$:h仅从最近命令的最后一个参数扩展目录

!!和!$可以替换为任何有效的扩展。

切分

CodeDescription
!!:n展开最新命令的第n个指令 （0是命令;1是第一个参数)
!^展开最新命令中的第一个参数
!$展开最新命令的最后一个指令
!!:n-m展开最新命令的指定范围的指令
!!:n-$展开最新命令的第n到最后一个指令

!! 可以替换为任何有效的扩展， !cat, !-2, !42, etc.

python hello.py >> output.txt  # 标准输出到（文件），追加

python hello.py 2>&1           # 标准错误输出到 标准输出
python hello.py 2>/dev/null    # 标准错误输出到（null）
python hello.py &>/dev/null    # 标准输出和标准错误输出到（null）

python hello.py < foo.txt      # 将 foo.txt 标准输入到 python
